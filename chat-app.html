Chat app

This application enable people to connect and chat through the web. Thanks to websockets, the app enables chating in real time without reloading the web page. With in-app notifications

+ 1

Tools

Frontend
  • NextJS, ReactJS, Redux, Tailwind CSS, Apollo Client (GraphQL), React Hook Form
Backend
  • NestJS, NodeJS, Prisma (Postgres SQL), Apollo Server (GraphQL), Redis, Jest
Other tools
  • AWS EC2, Route 53, Docker

How ?

Redux for in-app notifications (errors, friend request, etc...)

Redis to chat in real time between users, and for notifications

Docker for testing and local development

Amazon Web Service (EC2, Route 53) for deployment

GraphQL as API and real time communication with Websockets
